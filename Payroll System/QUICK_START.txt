BEFORE RUNNING:
1. Install MySQL Server 8.0.46 (from MySQL Installer)
2. Intall MySQL Workbench 8.0.46
2. Create database: "payroll"
3. Import database_schema.sql using MySQL Workbench

TO RUN APPLICATION:
1. Double-click: PayrollSystem.exe
2. Login with: admin / admin
3. Start adding employees!

IMPORTANT:
⚠️  All .dll files MUST be in same folder as .exe
⚠️  MySQL Server must be running
